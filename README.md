# Termory(terminar - memmory): A Command Line Tool

Termory is a simple tool for building command line utilities

## Quick Start
- Install ruby
and
- gem install termory
### Configuration
1. Init data
```sh
termory db -c
```
2. Setting your alias

```sh
termory config alias.xxx "your config"
```

3. Use your alias

```sh
termory exec xxx
```
